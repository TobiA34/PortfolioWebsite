Unit 2 - Web Design and Development

Introduction to web development and design

In this unit I will be learning how to build a website by creating a e-portfolio website which will be using stylesheets to style the way the website looks. This will be done by creating a CSS file which will have all the styles.

Week 1- Introduction to Unit and Html

In the first week of this unit I learned how to set up a html file by creating a folder to store the html file inside. I also learned the syntax to set up the html skeleton. I also learned how to use a h2 tag,br tag, img tag, a tag, ul tag. This can be seen in the pictures on the left.

Week 2- Introduction to Cascading Style Sheets

In the second week of this unit I learned how to set up a cascading style sheet file which is really important. The reason why CSS is important to use is because it allows me to style a website for example, in the pictures I have styled a mmu website and positioned the elements in different parts of the page.

Week 3- CSS Layout and Styling Menus

In the third week of this unit I have learned how to style a menu by using CSS. For example, I have set a red background for the menu and I have used display inline block to position the links in a line. I have used margin left to add space between each links so it looks nice on the web page.

How i feel about the course

I feel like web design and development is really fun because I get to be creative by building different websites in different layouts. I feel like this course will be really useful in the future because when I am applying for a job I can use these techniques to build my own e-portfolio websites.
