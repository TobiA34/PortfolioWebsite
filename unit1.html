Unit 1 - Programming

Introduction to programming

In this unit, I will be learning how to program. For example, in the first year, I will be learning the basics of programming such as variables, loops if statements and procedures. The program that I will be using to code is processing and each week I will be doing different exercises to improve my coding skills.

Week 1-Sequence and Variables

In the first week of this course I have learned about variables and expressions, drawing commands such as line, point, ellipse. I also learned how to use variables in my code and I also learned how to use a sequence. In the picture on the left I have used the variables x and y to determine the position of the different parts of the stick man.

Week 2- loops

In the second week of this course, I have learned how to use loops. I have made a loop to make this pattern by setting a size for the canvas and there is a variable called I and as long as the I value is less than 5 then it will draw the pattern but once it reaches 5 then it will stop executing the code.

Week 3-Procedures

In the third week of this course I have learned about procedures and i have created a procedure called grid and it has all of the code for the grid. I also created a procedure called noughts which has the course stored inside. Then i have created a procedure called setup and inside this i have called the procedure so that the grid and noughts can be drawn.

How i feel about the course

I feel like programming is fun because there is a lot of problem solving involved in this course and I like a challenge for example week 1 when I had to draw a stickman was really fun to do. I had to break the solution into small steps before bringing the whole stick man together. I think this course was a bit difficult at the start but as the weeks progressed then it started to become easier to do. I feel like I will learn a lot from this course in the end which I can apply in the workplace.
