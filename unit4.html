Unit 4 - Computer Systems Fundamentals

Introduction to Computer Systems Fundamentals

In this unit, I will learn about the fundamental principles and mathematics underpinning the design and construction of Computer Systems and it covers the Digital logic and Boolean Algebra: Digital logic gates and circuits.

Week 1- Digital gates and Combinations File

In week 1 I have learned about Digital gates and Combinations and how to create circuits in Logisim. I have learned about the different Boolean notation for each of the gates. Then I learned about more complex circuits which require more thinking to do. In this section I have completed exercise 1.4. on the left there is an example of one of the exercises and it has its truth table next to it.

Week 2- Combinational Logic Circuits - Boolean Sums of Products Notation File

In week 2 I have learned about Combinational Logic Circuits - Boolean Sums of Products Notation File and how to do the sum of products of an expression. Then I learned about more complex sums which require more thinking to do. In this section I have completed exercise 2.5. on the left there is an example of one of the exercises.

Week 3- Combinational Logic Circuits II

In week 3 I have learned about Combinational Logic Circuits II which is mainly about the ALU(Arithmetic Logic Unit). In the exercise I had to fill out a table to do with the values of the input switches are write down the correct operation. On the left there is an example of the work that I have done.

How i feel about the course

I think course is a really useful course because it teaches me how do different logic. I can apply this logic into my programming later in the future because I might come across a section where I need to use this logic to create a algorithm. At the start of the unit I felt like the unit was a bit hard but as the week goes on, I am getting used to the unit.
